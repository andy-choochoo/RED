File and Directory Description
-The directory data includes three public datasets we used.
-The script *SSR.py* (Search for Similar References) corresponds to the Multimodal Fusion Layer Self-Supervised Optimization; the directory *SSR* is used to save the results of this module.
-The script *EA.py* corresponds to the Experience Acquisition module; the directory *EA* is used to save the results of this module.
-The scripts *QDC.py* and *CA.py* correspond to the two layers of the Multi-Agent Dynamic Debate Protocol respectively; the directory *results* is used to save the results of these two layers.
